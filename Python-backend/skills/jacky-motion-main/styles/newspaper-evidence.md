# 报纸剪贴纪实风 (newspaper-evidence)

适用：新闻事件、历史、政策、人物、社会议题、案件调查。
气质：证据、档案、纪录片。观众觉得自己在看一组整理过的调查材料。

## 视觉硬规则

1. 剪报边缘必须有不规则感：撕纸边、纸纤维、胶带、错位阴影中至少出现 2 种，不做干净矩形白卡。
2. 红色标注框尺寸必须贴合被标注内容：词语用短下划线或小红圈，标题行用贴合短框。标注区域超过素材 45% 宽度时改用红圈/下划线/短箭头。
3. 色板仅限黑/白/米色 + 一种强调色（黄色荧光笔或红色批注），不可多色。
4. 剪报必须包含来源/标题/日期/引用/证据槽中的至少 2 个，不是空白卡片。
5. 每屏手绘装饰（箭头/圈注/波浪线）最多 3 个，且必须指向信息焦点。

## 组件库

**容器**：`.clipping` / `.clipping.torn-edge` / `.clipping.no-tape`
**文字**：`.kicker`（红色标签）/ `.headline`（衬线大标题）/ `.headline-sans` / `.sub-text` / `.body-text`
**强调**：`.highlight-mark`（黄色荧光笔）/ `.accent-word`（红色强调词）/ `.red-annotation`（红框标注）/ `.red-circle` / `.pencil-arrow` / `.red-line-through`（红色删除线）/ `.stamp`（印章）
**元信息**：`.source-label` / `.date-stamp`（等宽字体）/ `.evidence-number`（#1, #2...）
**辅助**：`.contra-vs`（竖向对比符）/ `.tl-line`（时间线虚线）/ `.figure-frame` / `.figure-placeholder`

## 运动个性

1. 主运动来自纸张滑入/标注划过/盖章/镜头推近，不来自弹跳或卡片漂浮。
2. 素材入场要有剪贴感：轻微旋转、纸张阴影、撕纸边、胶带固定。
3. 证据堆叠必须按口播顺序逐个出现，不能一开始铺满全屏。

## 风格禁忌

1. 整段涂黄——高亮只标 1-2 个短语。
2. 完美矩形截图卡片（没有剪贴感）。
3. 横跨半屏的长红框。
4. 只有静态剪报堆叠没有证据推进动作。
