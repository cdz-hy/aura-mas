# 内置风格

## apple-tech-gradient：苹果科技渐变演示风

适合 AI 工具、产品概念、抽象机制和 keynote 式能力展示。

关键词：纯黑空间、毛玻璃、克制渐变、模块错层、慢推、概念揭示。

避免：紫粉渐变、过度霓虹、随机粒子、万能三列卡片。

## editorial-magazine：杂志信息图风

适合深度知识、文化商业洞察、方法论、研究解读和人物故事。

关键词：强网格、大留白、跨页感、编号推进、引用落位、编辑版面。

避免：PPT 式 bullet、卡片堆叠、全居中排版、没有裁切逻辑的图片展示。

## finance-studio-cards：财经演播室卡片风

适合财经、商业、公司分析、政策和产业结构类内容。

关键词：专业信息大屏、主数字、关系路径、背发光卡片、节点逐段点亮。

避免：扁平深色卡片、所有节点同时出现、过度圆角 SaaS 风。

## newspaper-evidence：报纸剪贴纪实风

适合新闻事件、历史、政策、人物、社会议题和调查类内容。

关键词：剪报、胶带、撕纸边、红色批注、证据槽、档案感。

避免：完美矩形白卡、大面积涂黄、横跨半屏的长红框、静态素材堆叠。
