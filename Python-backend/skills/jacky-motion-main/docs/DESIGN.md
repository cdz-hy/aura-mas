# 设计理念

Jacky Motion 的目标不是生成一个漂亮网页，而是把口播稿变成更容易被理解、被录制、被复用的信息动画。

## 1. 信息表达驱动一切

先判断脚本本身能不能讲清楚，再决定画面。画面布局服务信息层次，动画服务表达节奏。

## 2. 每个镜头必须有视觉动词

视觉动词是画面动作和信息结构之间的桥梁。例如：

- 高亮：让观众知道现在看哪里。
- 连接：解释两个对象之间的关系。
- 对比：让差异被看见。
- 展开：把抽象概念拆成层级。
- 流动：展示流程或传递路径。
- 堆叠：让证据或层级逐步累积。

没有视觉动词的镜头，通常只是静态排版。

## 3. 文字必须短于口播

屏幕文字只承担“提示”和“定格”的作用，不复述整段口播。观众听口播获得细节，看屏幕抓住结构。

## 4. 单片单风格

一条视频只使用一种主风格。风格混搭会让观众把注意力放在画面变化上，而不是信息本身。

## 5. 4 段式镜头编排

每个节拍默认按 4 段设计：

1. 主视觉入场：建立画面主体。
2. 画面重构：布局变化或新元素加入。
3. 关键词强调：重点信息视觉突出。
4. 标注落位：辅助信息就位，停在可读构图。

这能让录屏时每次点击都进入一个完整、可理解的动画段落。
